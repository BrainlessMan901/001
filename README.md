# 자바 스윙을 통해 만든 점프게임

---

## 도움말

1~6번으로 블럭/지우개/시작,종료 지점 설정
  
enter키로 테스트, backspace로 테스트 중단
  
a, d로 이동, w로 점프
  
종료 지점에 도달 할 시 좌측 상단의 저장 버튼으로 맵 저장
  
좌측 상단의 폴더 버튼으로 저장한 맵 불러오기
